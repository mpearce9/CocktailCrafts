## Project setup

## Navigate to frontend folder
```
cd frontend
```
### Install dependencies
```
npm install
```
### Build frontend
```
npm run build
```
### Reroute to root directory then to the server
```
cd ../server
```
### Install dependencies
```
npm install
```
### Run server with backend and frontend together
```
npm start
```
